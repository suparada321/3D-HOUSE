# 3D Models Directory

วางไฟล์โมเดล 3D (.glb) ที่นี่:

- `house-a.glb` - Type A: Modern Minimal
- `house-b.glb` - Type B: Contemporary Luxury  
- `house-c.glb` - Type C: Tropical Villa

## แหล่งโมเดล 3D ฟรี:

1. **Sketchfab** - https://sketchfab.com/
   - ค้นหา "house", "modern house", "villa"
   - ดาวน์โหลดเป็น glTF (.glb)

2. **Poly Haven** - https://polyhaven.com/
   - โมเดลคุณภาพสูง พร้อม textures

3. **TurboSquid** - https://www.turbosquid.com/
   - มีโมเดลฟรีและเสียเงิน

## Tips:
- ขนาดไฟล์ควรไม่เกิน 10MB
- Polygon count ไม่ควรเกิน 100k triangles
- ใช้ Draco compression ถ้าทำได้

## หมายเหตุ:
ถ้าไม่มีไฟล์โมเดล ระบบจะใช้ Fallback Model (บ้าน 3D แบบง่าย) แทน
